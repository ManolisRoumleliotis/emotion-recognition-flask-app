# Emotion Recognition & LLM Web Application

## Περιγραφή Έργου
Η παρούσα εφαρμογή αποτελεί ένα έξυπνο web σύστημα που συνδυάζει
τεχνικές υπολογιστικής όρασης και τεχνητής νοημοσύνης για την
αναγνώριση συναισθημάτων από εικόνες προσώπου και την παραγωγή
φυσικής γλώσσας μέσω Large Language Model (LLM).

Η εφαρμογή αναπτύχθηκε στο πλαίσιο του μαθήματος Artificial Intelligence
και υλοποιεί ένα ολοκληρωμένο pipeline από την ανάλυση εικόνας
μέχρι τη δημιουργία περιγραφικού κειμένου.

---

## Λειτουργίες
- Upload εικόνας και αναγνώριση συναισθήματος
- Ανίχνευση προσώπων με bounding boxes και confidence scores
- Ανάλυση συναισθήματος από webcam σε πραγματικό χρόνο
- Οπτικοποίηση κατανομής συναισθημάτων:
  - Bar chart
  - Line chart
  - Pie chart
- Παραγωγή περιγραφικού κειμένου μέσω Large Language Model (LLM)
- Web εφαρμογή βασισμένη στο Flask

---

## Τεχνολογίες που χρησιμοποιήθηκαν
- Python
- Flask
- OpenCV
- FER (Facial Expression Recognition)
- NumPy
- Matplotlib
- Ollama (Local Large Language Model)

---

## Αρχιτεκτονική Συστήματος
Η εφαρμογή ακολουθεί μια αρθρωτή αρχιτεκτονική με διακριτά στάδια:

1. Εισαγωγή εικόνας μέσω upload ή webcam frame
2. Ανίχνευση προσώπου και αναγνώριση συναισθήματος
3. Υπολογισμός dominant emotion και confidence scores
4. Συλλογή αποτελεσμάτων και δημιουργία οπτικοποιήσεων
5. Δημιουργία prompt προς το LLM
6. Παραγωγή φυσικής γλώσσας και εμφάνιση αποτελέσματος στο UI

---

## Εκτέλεση Τοπικά
Η εφαρμογή μπορεί να εκτελεστεί τοπικά σε περιβάλλον Python.

Απαιτείται:
- Python 3.x
- Virtual environment
- Εγκατάσταση των βιβλιοθηκών που περιλαμβάνονται στο `requirements.txt`

Η εκτέλεση πραγματοποιείται μέσω του αρχείου `app.py`.

---

## Διαμόρφωση LLM
Για την παραγωγή φυσικής γλώσσας χρησιμοποιείται τοπικά
φιλοξενούμενο Large Language Model μέσω του Ollama.

Το σύστημα είναι συμβατό με οποιοδήποτε διαθέσιμο open-source
μοντέλο (π.χ. Mistral, Phi-3), ανάλογα με τους διαθέσιμους πόρους
του συστήματος.
